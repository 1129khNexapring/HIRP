## 1. 개요
HIRP = High + ERP<br>
높은 기능성, 높은 사용성을 갖춘 ERP.
<br><br>

## 2. 설계의 주안점
기본에 충실하며 뛰어난 품질의 ERP를 제작할 것.<br>
기업 및 개인 사용자에게 최대의 편리를,<br>
개발팀 내에선 유지보수와 확장성이 좋도록 프로그래밍.<br><br>
